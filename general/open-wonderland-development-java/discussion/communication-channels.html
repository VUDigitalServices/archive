Bosco A

There's a bunch of Web 2.0 technologies we can leverage all around for online collaboration. We can pick what we need from:
(1) gmail chat/skype - voice communication/desktop sharing
(2) Teamviewer - presentations/desktop sharing
(3) Google sites - project template - setting up individual projects, wiki, sharing working docs (temporary)
(4) Youtube - setting up and sharing playlists of interest, creating short instructional videos on basic and advanced concepts. See .. commoncraft.com for some low tech videos on web 2.0 technologies. http://www.youtube.com/results?search_query=in+plain+english&aq=f
(5) Social bookmarking using either
...(a) Google notebook http://www.google.com/googlenotebook/tour1.html
...(b) Delicious
...(c) Memonic
(6) Blogs
(7) Twitter for more intermittent communication about how the team members are doing with projects, learning experience etc.
(8) Google Reader for setting up feeds from various websites of interest
(9) Social networking - Facebook - getting feeds from Wonderland, etc.

Of course the forum, OWL groups would be the main channels for P2PU course and OWL related communication.
