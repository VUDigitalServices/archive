One of my course members has pointed out an issue with the Course Materials page yellow font on a charcoal background.  He finds this quite hard to read -- is there a way for me to change this or is there a browser setting that he can use to change the font or background colors?   Thanks for any help on this!!

I think it's an overall style depending on which school. I used to be able to switch it out by changing the url but I think that's been fixed. Example: instead of http://p2pu.org/general/find-author-and-share-open-textbooks I could load up the webcraft style with http://p2pu.org/webcraft/find-author-and-share-open-textbooks

I know you can change CSS with browser plugins and turn off style with them as well. One of my favorite ways to manipulate what is shown is http://chrispederick.com/work/web-developer/

Depending on his browser, he can also set it there too. In Firefox, under Tools -> Options -> Content click advanced, and uncheck 'Allow pages to choose their own' and it should be the browser default. He can make that be whatever he would like. He'll need to switch back to 'Allow' if he wants other sites to show their styling.
