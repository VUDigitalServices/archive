WCAG 2.0 POUR Matrix

Thu, 2011-02-10 07:56

The 4 principles of Web Accessibility in WCAG 2.0 are:

Principle Indicator Disability Category
Perceivable images need text alternatives visual
  contrast ratio visual
  captions for audio hearing
  content flicker visual, cognitive
Operable    
  keyboard access to all functions physical
  user control for timed responses physical, cognitive
  heading structure for navigation & content retrieval physical, visual
Understandable    
  Language ID visual, cognitive
  Content, system operates predicatably cognitive
  Input errors can be avoided or corrected physical, cognitive
  visual clarity, readability visual, cognitive
Robust    
  Content Delivery is independent of single technology all disabilities
  assistive technology compatible all disabilities

There is a good checklist for web accessibility available on the WebAIM website - http://www.webaim.org/standards/508/checklist
