Beginning Game Development with HTML 5

Course organiser: Jono Xia
About the Course Organiser: 

I'm Jono Xia, I work in user research at Mozilla and I am also a hobbyist game developer.  I helped out with the Mozilla Labs 2010 Game On contest which encouraged people to develop games using open web standards.  I want to encourage more people to develop games for the open Web, as opposed to writing iPhone apps or Flash games or other proprietary technology platforms, because I think having more games that will run on any Web-capable device is good for gamers, good for game developers, and good for the Web!
 

I'm teaching myself a lot of HTML 5, canvas, Javascript, and networking techniques in order to develop a game project of my own, so I figure, why not share what I'm learning and encourage others to express their ideas in the same medium?
