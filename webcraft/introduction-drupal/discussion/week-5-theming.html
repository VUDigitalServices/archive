Week 5: Theming

Benji Fisher on Thu, 2010-10-21 15:15
Go back to: General discussion

So far, there have been no tasks set for Week 5, which is supposed to cover theming.  The syllabus mentions the "Fusion/Skinr theme set."  Does anyone have some link appropriate to that?  What can you do with Fusion/Skinr, how do you get started?

A good way to get started, if you are not planning to design a theme from scratch, is to choose an existing theme and then start tweaking it.  As usual, there are problems with hacking the distributed files, so a good approach is to create a sub-theme.  I recommend Subtheming Quick and Dirty as a way to get started.  The first third of this article is pretty easy.  The rest of it gets into some more advanced things you can do with your theme.  Even if you do not try to follow the later examples, you might want to read them quickly to get an idea of the possibilities.  Drupal themes give you a lot of control over your site that goes way beyond the "presentation layer."

I am designing a theme for another site.  I would like to add something like a second site logo:  let the user choose a file in the theme configuration page, then include it in the page template file.  If anyone can point me to a suitable tutorial, I would appreciate it.  If not, I will search drupal.org myself ...
