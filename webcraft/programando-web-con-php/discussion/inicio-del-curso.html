Inicio del curso

Martin Olivera on Wed, 2011-01-26 01:29
Go back to: General discussion

Hola a todos

lean con atencion todo este mensaje, es la introduccion al curso "programando web con PHP"

-----------------------------------------------------
COMUNICACION EN FOROS

Vamos a utilizar los foros, para organizarnos, cada semana voy a abrir una nueva conversacion con los temas a tratar, donde se pueden hacer las preguntas, observaciones y comentarios sobre temas de esa semana.

Ademas del hilo semanal en FORUM voy a abrir tres hilos mas, que vamos a usar durante todo el curso, a saber:

A - Infraestructura, herramientas: dedicado a como instalar, probar, con que programas desarrollar, problemas de configuracion e instalacion, etc. No son temas que tienen que ver estrcitamente ni con PHP ni con los temas del curso, pero son necesarios para poder aprovecharlo, les ruego que usen EXCLUSIVAMENTE ESE HILO para estos temas, y no el hilo semanal

B - Lenguaje PHP, sintaxis y diversos usos: este hilo tendra informacion y consultas relacionadas con la forma correcta de programar en PHP, especificamente dedicado a la sintaxis y a la organizacion de archivos, funciones, etc. y no será especificao de cada semana. Tambien es el lugar para preguntar dudas sobre como se escribe o cual es la mejor forma de hacer algo, que no este relacionado con el tema del curso semanal

C - Varios. Una lista para hacer preguntas, observaciones y propuestas "descolgadas" siempre debe haber, cuando tengan algo que compartir, que no sea del tema de la semana, ni sobre infraestructura o herramientas, ni sobre el lenguaje PHP propiamente dicho, USEN ESTA LISTA.

------------------------------------------------------
ESPACIO DE EJEMPLOS Y PRUEBAS

Para probar PHP hace falta disponer de un servidor, todos pueden instalarse uno localmente en su computadora o pueden disponer tal vez de algun servicio de hosting donde probar sus ejercicios (detalles sobre esto, en el FORUM - Infraestructura

Adicionalmente, para quienes quieran hacer pruebas y no dispongan, o tengan problemas para hacer andar su servidor, o quieran compartir sus pruebas y ejercicios con todos, monte un espacio de pruebas en
http://nodosol.org/p2pu

los ejemplos de cada semana se podran descargar de ahi (los detalles y data de los ejemplos, estaran en el hilo semanal correspondiente) y ademas podran ingresar con el usuario de cada uno - el mismo con el que estan registrados en P2PU - y subir archivos y probar como ejecutan. Sean cuidadosos y criteriosos, los archivos una vez subidos ahi no se pueden borrar y el espacio es limitado, por favor no se extralimiten subiendo cosas pesadas, gracias!
