en el directorio de apache -> sites available -> default (edita las lineas que te indico):
1.- DocumentRoot /directorioactual/
por
DocumentRoot /tudirectorio/
2.-

<Directory /directorioactual/>
Options Indexes FollowSymLinks MultiViews
AllowOverride None
Order allow,deny
allow from all
</Directory>

por
<Directory /tudirectorio/>
Options Indexes includes FollowSymLinks MultiViews
AllowOverride None
Order allow,deny
allow from all
</Directory>

, después de los cambios reinicia apache.
