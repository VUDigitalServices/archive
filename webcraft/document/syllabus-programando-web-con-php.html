Programando web con PHP

coordina: Martín Olivera (martin.olivera@nodosol.org)

curso inicial para aprender a programar la web, utilizando el lenguaje PHP

inicia 26 de enero de 2011

antes de iniciar, se recomienda leer:

  • Definición de World Wide Web (Wikipedia) http://es.wikipedia.org/wiki/WWW
  • Definición de HTML (Wikipedia) http://es.wikipedia.org/wiki/HTML
  • Arquitectura cliente-servidor http://es.wikipedia.org/wiki/Cliente_servidor

Semana 1: Intrducción
Como funciona la web.
Arquitectura cliente-servidor en la web.
HTML. Navegadores y  servidores web.
Montar tu propio servidor web. Probando PHP

Semana 2: Lenguaje PHP
Sintaxis. Separadores. Uso de funciones.
Variables, números y textos. Operadores.
Estructuras de control: condicionales y bucles.

Semana 3: PHP embebido en HTML
Modelos de programación PHP.
Uso de formularios. Pasaje de parámetros GET y POST
Uso de múltiples archivos. Aplicaciones web.

Semana 4: Arrays.
Crear, recorrer y operar con arrays.
Arrays predefinidos.

Semana 5: Objetos en PHP
Clases, objetos. Atributos y métodos. Restrictores de acceso.
Uso de objetos existentes. Programación de clases reusables.

Semana 6: PHP y bases de datos.
SQL. Acceso desde PHP

Semanas 7 y 8: Integrando conocimientos
Creación grupal de un CRUD (ABMC) con PHP y base de datos

Semana 9: Aplicaciones libres en PHP
Opciones. Ventajas y desventajas.
Pruebas sobre CMS libres hechos en PHP

Semana 10: Por dónde seguir
Nociones de Javascript. AJAX.
Frameworks libres en PHP.
