Recommended reading

Fri, 2011-03-04 16:34

If you could only read three books after this course, read these ones:

- Bullet Proof Web Design by Dan Cederholm
- Don't make me think by Steve Krug
- How to Write Great Copy for the Web by Donna Spencer OR Letting Go of The Words by Janice Reddish

If you could only read three articles after this course, read these ones:
- Color Theory: http://www.smashingmagazine.com/2010/01/28/color-theory-for-designers-pa...
- The top 10 mistakes of web design http://www.useit.com/alertbox/9605.html
- How users read on the web: http://www.useit.com/alertbox/9710a.html

Design Theory (also known as: "the reason why it looks like this")

- Design Theory concepts: http://webdesign.tutsplus.com/category/articles/design-theory/?tag=basix
- Color Theory: http://www.smashingmagazine.com/2010/01/28/color-theory-for-designers-pa...

Creating web page elements
- http://webdesign.tutsplus.com/category/tutorials/site-elements/

CSS positioning
- CSS positioning 101: http://www.alistapart.com/articles/css-positioning-101/

Creating complete web page designs
- Bullet Proof Web Design by Dan Cederholm
- http://webdesign.tutsplus.com/category/tutorials/complete-websites/

Designing for iPad
- http://www.inspiredm.com/ipad-design/
- http://www.labnol.org/internet/test-websites-on-ipad/13368/
- http://blog.templatemonster.com/2010/05/14/ipad-web-design/
- http://mashable.com/2010/09/30/ipad-web-design/

Writing for the web
- Best short book: How to Write Great Copy for the Web
- Best complete book: "Letting go of the words: writing web content that works" by Janice Reddish

Usability
- Don't make me think by Steve Krug
- www.useit.com/alertbox/ - Alertbox colums by Jakok Nielsen

Usability Testing
- Rocket Surgery made easy by steve Krug

Creating page mock-ups
- http://www.balsamiq.com/

Remote Usability Testing tools - and other tools to improve your site (even if you only have a couple of participants, you can learn a LOT, so sign up for the free trials)
- Websort: online cardsorting tool - free limited account
- Optimal Work Shop: card sorting, tree testing (treejack) and clickable mock-ups (chalkmark - use your Balsamiq mock-ups!) - free limited account
- www.userfly.com - record what visitors do on your website - free limited account
- www.usertesting.com - like userfly, only people are screened to do certain tasks and they tell you what they are doing ( NOT FREE)
- Google Website Optimizer - test different variatons of a webpage
