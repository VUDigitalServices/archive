Summary

PHP is Open Source and Cross Platform, running on Unix, GNU/Linux, MS Windows, Apple Mac, AS/400 and more, and is often used with an OS-CP database and web server such as MySql and Apache. Thus PHP will work on the operating system and hardware that currently suits your applications and organisation.

PHP and Databases

  • Introduction to Programming Languages
  • PHP Language Basics: Syntax, Quotes, Comments, Data Types
  • PHP Variable Basics: Numbers, Constants, Variables, Strings, Arrays
  • PHP Control Flow Basics: Operators, Conditionals, Loops
  • PHP Functional Basics: Functions, Libraries, Advanced Arrays, File Handling
  • PHP Programming Basics: Best Practices, Debugging, Brief Introduction to HTML
  • PHP Web Basics: Brief Introduction to Server Communications, Sessions, Security
  • PHP Database Basics: Brief Introduction to MySQL/MariaDB/SQLite/NoSQL Database as it relates to Web Sites
  • PHP Applications Basics: Best Approaches to Creating Applications;
  • Course project

Publish

  • Where to publish your PHP project.
  • How to find co-wokers for your great idea.
  • Choosing a (free) license of your code.
  • Learn more about opensource licenses and Creative Commons.
  • How to contribute to OpenWeb using your PHP skills

PHP 5 is the current version of PHP and is what the course teaches.

This course is designed for those who want to learn the basics of the PHP language, and assumes some familiarity with web sites – what they are and the very basics of how they work – but no (or limited) programming experience. Web designers who want to become PHP Developers would certainly benefit from this course.

Main objective:
To learn the basics of the PHP programming language.

Prerequisites

  • A basic working knowledge of HTML is presumed.
  • Some conceptual knowledge of how web servers and web browsers (clients) interact using HTTP.
  • A computer or a hosting space where you can test your work.
  • A free IDE like Komodo Edit
  • Since this course will be in English, basic knowledge is required. (If anyone wants to translate materials to other languages, we can add more languages)
