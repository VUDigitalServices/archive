Ron Allison
Sat, 2011-01-29 19:55

The way I understand the [gallery] shortcode is it uses images attached the specific post or page you put the code in. So the pictures have to be attached to the page you are wanting to be the gallery. This may not work for what you are trying to do. Really it is a very simple solution, maybe not powerful enough to do what you want.

If you've uploaded images to your media are you can use those with the [gallery] shortcode. What you do is attach them to the post/page where you are using the [gallery] shortcode. You can do this in the media admin section. Just browse the pictures, then click attach. This assumes they aren't attached to another post/page. I do not think they can be attached to multiple pages. However, I'm still a noob at this so I could be wrong.

Hi Polly

If you installed using Fantastico you might want to update WordPress immediately (security release) - Fantastico is often a little behind the most current WP (e.g. 3.0.2 rather than the now current 3.0.4). Here's the information about the latest version of WordPress: http://wordpress.org/download/

But it's very easily updated from the Dashboard once you've logged in ( when you're logged in at e.g. pollysdomain.com/wp-admin/ ).

I can't help specifically with Filezilla (I use FireFTP) but you should be able to access your files via ftp, it might just be a question of changing the settings? And your hosting company (HostGator) should be able to help with that if you have problems.

Good luck and do post back, I'm sure there are plenty here who know FileZilla and/or Hostgator!
