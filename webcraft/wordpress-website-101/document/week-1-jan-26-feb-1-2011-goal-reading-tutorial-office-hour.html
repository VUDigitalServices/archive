Hi Everybody -

This is our Wordpress 101 agenda for the first Week (Jan 26 - Feb 1 2011)

 

1. Some of you already did this but if you haven't,

- Link to the Project Site (if exists)

 

2. Our main goal for this week is to get acquainted with the wordpress platform.

Install wordpress, get to know dashboard, install themes, plugins, write our first blog entry, set up permalinks, user roles etc.

 

What is wordpress?

Introduction

http://codex.wordpress.org/Introduction_to_Blogging

Some reading about wordpress and other content management system

http://www.theopensourcery.com/keepopen/2010/best-open-cms-wordpress-vs-joomla-vs-drupal/

 

3. Most of you already have a wordpress site or even a static website, as far as I know.

If project site doesn't exist, please create one by installing a fresh wordpress copy.

If you need assistance setting up, please refer to the following info.

(1) Easy way - If your hosting company supports wordpress install

http://www.siteground.com/tutorials/wordpress/wordpress_installation.htm (Txt Tutorial)

http://wordpress.tv/2009/02/09/one-click-self-installation-of-wordpress-with-hostican/ (Video Tutorial)

(2) DIY way - If your hosting company doesn't support wordpress install or if you wanna do it on your own.

http://www.siteground.com/tutorials/wordpress/wordpress_manual_installation.htm (Txt Tutorial)

http://www.youtube.com/watch?v=6nse3Gdtvs4 (Excellent Video Tutorial)

 

(3) If you don't have a hosting service, find a free hosting service before you install wordpress.

I personally haven't tried any of the service the link below suggests but it might be worth trying, if you just wanna set up a free hosting service for now.

http://www.free-webhosts.com/free-domain-hosting.php
