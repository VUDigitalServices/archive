Victor Ramirez
Mon, 2011-01-24 00:15

A blog aka web log is a type or part of a website. So, yes we are creating a wordpress blog and website in this course. Blogs will not replace websites because a blog is a website that does a specific task. Wordpress is an open source blogging platform that can be used as a content management system (CMS) for websites beyond blogs. Saying a website is more of a hassle than a website is completely subjective. A website can be a single static page (a menu and contact information for a restaurant) to something complex as Amazon.
